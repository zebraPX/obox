1.下载文件

js：依赖 jQuery / Dome 使用 jauery-1.8.3.min.js（支持以上版本）框架基础 "obox.js"

css： LESS / 框架基础两个less文件 "obox.less" （框架基础样式） "obox_set.less"( ui配置文件 ) ,ui配置结束后推荐使用编译软件 koala 最终页面引入编译生成框架基础样式 obox.css

打包下载obox


2.HTML基础(重点)

为了确保适当的绘制和触屏缩放，需要在 <head> 之中添加 viewport 元数据标签。

        <meta name="viewport" content="width=device-width, initial-scale=1, user-scalable=no, minimal-ui"/>

布局容器：页面所有内容必须包在.o_body > .o_main 内缺一不可 （除弹层内容,详情见 oPopup弹层模块）

     <div class="o_body">
        <div class="o_main">
              ...
        </div>
     </div>
